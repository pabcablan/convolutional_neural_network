# Convolutional Neural Network

## Dataset

This project uses the [Eye Diseases Classification dataset](https://www.kaggle.com/datasets/gunavenkatdoddi/eye-diseases-classification) from Kaggle.

### Setup Instructions

#### 1. Install Required Packages

```bash
pip install opendatasets pandas
```

#### 2. Configure Kaggle Credentials

1. Go to [Kaggle Account Settings](https://www.kaggle.com/settings)
2. Scroll down to the **API** section
3. Click **"Create New API Token"**
4. A `kaggle.json` file will be downloaded with your credentials

#### 3. Download the Dataset

The dataset will be automatically downloaded when you run the Jupyter Notebook. You will be prompted to enter your Kaggle username and API key (found in `kaggle.json`) on first run.
